Cable Driven Robot

Senior Design Project - Fall 2017

Overview

This project was developed for McMurough's senior design class of fall 2017. The Project was completed during the fall summer term and was successfully completed by Robert Aron, Taylor Cowdin, Patrick Tennyson, and Martin Martinez.

The project goal was to design and implement a robot with similar functionalities to a robot arm, but using cables as the drivers. The project takes X-Y-Z inputs from the user, then translate those points to a list of locations into a path that the end effector moves along.

About

Our project goal was to develop a system that would allow users to input paths that could run an end effector tool. After months developing requirements for the project, we set out with a model idea in mind with a large list of necessary tools that were needed for the completion of the project.

Many pieces of the project required us to take on new hats and learn new tools. None of the team members initially had experience with physically engineering and kind of robot, or what kind of standards there were in robotics development. In addition to robotics knowledge, we needed to learn modeling techniques to make the design cohesive. Many of the tools we believe would be readily available were not as easy to find as we initially believed. Many parts of the robot needed to be modeled and printed out for the specific needs in order to work properly with the system.

During the second semester of the interfaces, we had been developing became integrated. Many of these systems integrated well, but many also had bugs. Some of the requirements we had initially planned on having changed in order to fit what we needed in order to complete the overall goal of the project. After a few hickups in the development, the project was set on a clear goal of competition based on the timeline set before us.

The final developed project met all of the initial goals designed. The wire robot operates efficiently and accurately without any issues. It is prepared for continued development and uses in another project in upcoming semesters for senior design students at the University of Texas at Arlington.
